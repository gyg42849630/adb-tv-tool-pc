# ADB电视工具 - 简化版

一个专门用于管理电视设备的简化工具，专注于核心功能：APK安装和屏幕截图。

## 🚀 核心功能

- **设备连接管理** - 连接/断开指定IP的电视设备
- **APK安装** - 从电脑安装APK文件到电视
- **屏幕截图** - 实时获取电视屏幕截图
- **指令监控** - 显示所有ADB命令执行过程和结果

## 📦 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动程序

```bash
# 使用简化版界面 (推荐)
python adb_tv_tool/simplified_main.py

# 或者使用完整版界面
python adb_tv_tool/main.py
```

## 🎯 使用指南

1. **连接设备**: 在设备连接区域输入电视IP地址（如：192.168.1.100:5555）
2. **安装APK**: 切换到APK安装选项卡，选择文件并安装
3. **获取截图**: 切换到屏幕截图选项卡，点击获取截图按钮
4. **监控指令**: 在底部监控区域查看所有ADB命令执行详情

- 确保电视设备已开启USB调试模式
- 设备和电脑需要在同一网络
- 截图功能需要设备支持ADB screencap命令
