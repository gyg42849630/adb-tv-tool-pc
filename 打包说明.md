# 📦 ADB电视工具打包说明

## 🚀 打包目标

将Python版本的ADB电视工具打包成独立的Windows可执行文件(.exe)，无需安装Python环境即可运行。

## ?? 打包配置

### 使用工具
- **PyInstaller 6.17.0** - Python打包工具
- **UPX压缩** - 减小可执行文件体积

### 打包参数
```bash
pyinstaller --onefile --console adb_tv_tool/simplified_main.py --name "ADB_TV_Tool"
```

### 关键配置
- **--onefile**: 打包成单个exe文件
- **--console**: 显示控制台窗口（用于调试日志）
- **--name**: 指定输出文件名
