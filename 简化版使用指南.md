# 📱 ADB电视工具 - 简化版使用指南

## 🎯 核心功能

简化版专注于您需要的4个核心功能：
1. **设备连接管理** - 连接/断开指定IP的电视设备
2. **APK安装** - 从电脑安装APK文件到电视
3. **屏幕截图** - 实时获取电视屏幕截图
4. **指令监控** - 显示所有ADB命令执行过程和结果

## 🚀 快速开始

### 1. 启动程序
```bash
python adb_tv_tool/simplified_main.py
```

### 2. 连接设备
- 在"设备连接管理"区域输入电视IP地址
- 默认已填写：`192.168.1.100:5555`
- 点击"连接设备"按钮
- 状态栏显示"设备状态: 已连接"

### 3. 功能使用

#### 📦 APK安装功能
1. 切换到"APK安装"选项卡
2. 点击"浏览"选择APK文件
3. 点击"安装APK"开始安装
4. 监控进度条和日志区域

#### 📸 屏幕截图功能
1. 切换到"屏幕截图"选项卡
2. 点击"获取截图"获取当前屏幕
3. 截图显示在预览区域
4. 点击"保存截图"保存到本地

## 🔧 界面说明

### 设备连接管理区域
- **设备IP输入框**: 输入电视设备IP地址
- **连接设备**: 建立ADB连接
- **断开设备**: 断开当前连接
- **设备状态**: 实时显示连接状态

### APK安装选项卡
- **APK文件选择**: 浏览和选择APK文件
- **安装进度条**: 显示安装进度
- **安装按钮**: 执行安装操作

### 屏幕截图选项卡  
- **获取截图**: 获取当前电视屏幕
- **保存截图**: 保存截图到本地文件
- **预览区域**: 显示截图内容

### ADB指令监控区域
- **实时日志**: 显示所有ADB命令执行
- **清空日志**: 清除当前日志
- **自动滚动**: 始终显示最新信息

## 🔍 故障排除

### 常见问题

**连接失败**
- 检查IP地址是否正确
- 确认电视USB调试已开启
- 确保设备在同一网络

**APK安装失败**
- 检查APK文件是否完整
- 确认设备存储空间充足
- 查看错误日志获取详细信息

**截图失败**
- 检查设备显示状态
- 确认ADB权限正常
- 重新连接设备尝试

### 调试技巧

**查看详细日志**
- 所有操作都在指令监控区域显示
- 包含完整的命令和返回结果
- 便于问题诊断

**重新连接**
- 断开后重新连接可解决临时问题
- 状态栏实时反映连接状态
