# 增强版 ADB 电视 APP 安装工具

一个采用苹果风格界面设计的专业电视应用管理工具，支持屏幕投屏预览、应用管理、批量安装等功能。

## 功能特性

### 🖥️ 苹果风格界面
- macOS/iOS 设计语言
- 磨砂玻璃效果
- 流畅过渡动画
- 极简主义设计

### 📺 核心功能
- **设备连接**: 自动扫描/手动连接电视设备
- **屏幕投屏**: 实时预览电视屏幕（流畅/高清模式）
- **应用管理**: 带图标的应用列表，支持批量卸载
- **APK安装**: 批量安装APK文件
- **智能检测**: 自动环境检测和错误处理

### 🔧 技术特性
- 基于PyQt6的现代化UI框架
- 多线程ADB操作
- 智能错误恢复机制
- 跨平台支持（Windows/macOS）

## 项目结构

```
adb-tv-tool/
├── src/
│   ├── main.py                 # 主程序入口
│   ├── ui/                    # 界面组件
│   ├── core/                  # 核心功能模块
│   └── utils/                 # 工具类
├── resources/                 # 资源文件
├── requirements.txt           # 依赖包
└── README.md                  # 项目文档
```

## 快速开始

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python src/main.py
```
